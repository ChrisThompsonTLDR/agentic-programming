# Laravel Ingest

**Laravel Ingest is a package that provides a clean, expressive API for ingesting and processing data from external sources into your Laravel application.**

## Key Features
- Fluent, chainable API for defining ingestion pipelines
- Supports multiple source types (files, APIs, streams)
- Configurable transformations and mappings during ingest
- Works with Laravel's queue system for background processing
- Publishable config for customising behaviour

## Installation

```bash
composer require zappzerapp/laravel-ingest
```

Optionally publish the config:

```bash
php artisan vendor:publish --tag=laravel-ingest-config
```
